Tipografias tags h1 até h6

As tags h1 até h6 servem para a tipografia do título de um determinado assunto, pois deixa o título em negrito, sendo h1 a tipografia maior e h6 a tipografia menor, das utilizadas nesta página.

Para que serve a tag 'p'?

O 'p' serve para criar um novo parágrafo.

Já a tag 'mark' iremos utilzar quando?

A tag 'mark' é utilizada para marcar/grifar uma determinada palavra, frase ou texto desejado.

Aprendendo sobre as tags 'i', 'u' e 'strong'

A tag 'i' serve para deixar o texto em italico.

A tag 'u' serve para deixar o texto sublinhado.

Já a tag 'strong'serve para deixar o texto em negrito.

E sobre a tag 'blockquote'??

Agora vamos abordar sobre as tags 'ol', 'ul', 'li'

A tag 'li" deve ser utilizada tanto para listas ordenadas 'ol', quanto para listas não ordenadas 'ul', onde deve estar abraçando o item

A tag 'ol' serve para fazer uma lista ordenada, ou seja, em ordem numérica crescente, conforme exemplo abaixo:

  1. Casa
  2. Carro
  3. Moto
  4. Terreno

A tag 'ul' serve para fazer listas não ordenadas, ou seja, ao invés de numerar os items aparecerá bolinhas (marcadores).

  • Mesa
  • Cadeira
  • Armário
  • Estante
